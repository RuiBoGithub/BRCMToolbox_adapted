1. Read the EnergyPlus IDF and parse zones, surfaces, constructions, materials, windows, and schedules.
2. Convert these into a common normalized building-data structure.
3. Derive shared physical quantities: areas, volumes, U-values, thermal masses, ventilation/infiltration, and boundary conditions.
4. Pass the normalized data to a `generate_5R1C()` function to aggregate parameters into the fixed 5R1C topology.
5. Pass the same normalized data to a `generate_7R2C()` function to aggregate parameters into the fixed 7R2C topology.
6. Pass the same normalized data to `generate_BRCM()` to construct the detailed multi-node RC network.
7. Apply identical weather, gains, controls, and initial conditions to all three models.
8. Simulate and compare temperatures, energy use, calibration performance, and ECM-saving predictions.
